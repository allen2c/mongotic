# MGT-002: ScalarResult.count()

## Description

Add `.count()` to `ScalarResult` that returns the number of documents matching
the statement's filters, without fetching documents into Python.

## Expected API

```python
count = session.scalars(select(User).where(User.age > 18)).count()
```

## Implementation Notes

- Use `collection.count_documents(filter)` on the collection
- Does NOT apply limit/offset (counts total matches)
- `ScalarResult` must hold a reference to the `Select` stmt and session/engine
  to issue the count query

## Acceptance Criteria

- [ ] `.count()` returns `int`
- [ ] Respects `.where()` filters
- [ ] Does not load documents into memory
- [ ] Test added to `tests/test_orm.py`
