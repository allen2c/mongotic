# MGT-017: Remove pyassorted dependency and inline required utilities

## Description

`pyassorted` is listed as a production dependency (`pyproject.toml` →
`[tool.poetry.dependencies]`) but is only used in tests. The two functions
actually consumed are trivial and can be inlined without any third-party package.

### Functions in use

| Import | Used in | Replacement |
|--------|---------|-------------|
| `from pyassorted.string import rand_str` | `tests/test_orm.py`, `tests/test_model_filed_operators.py` | stdlib `random` + `string` |
| `from pyassorted.datetime import aware_datetime_now` | `tests/test_orm.py`, `tests/test_model_filed_operators.py` | `datetime.now(timezone.utc)` |

## Steps

### 1. Create `tests/helpers.py`

Implement the two utilities using the stdlib only:

```python
import random
import string
from datetime import datetime, timezone


def rand_str(length: int = 10) -> str:
    chars = string.ascii_letters + string.digits
    return "".join(random.choice(chars) for _ in range(length))


def utc_now() -> datetime:
    return datetime.now(timezone.utc)
```

### 2. Update `tests/test_orm.py`

- Remove `from pyassorted.datetime import aware_datetime_now`
- Remove `from pyassorted.string import rand_str`
- Add `from tests.helpers import rand_str, utc_now`
- Replace `default_factory=aware_datetime_now` → `default_factory=utc_now`

### 3. Update `tests/test_model_filed_operators.py`

- Same import swap as above

### 4. Remove from `pyproject.toml`

```toml
# delete this line from [tool.poetry.dependencies]:
pyassorted = "^0.7.5"
```

### 5. Regenerate lock / requirements

```bash
poetry lock
poetry export -f requirements.txt --without-hashes -o requirements.txt
poetry export -f requirements.txt --without-hashes --with dev -o requirements-dev.txt
```

## Acceptance Criteria

- [ ] `tests/helpers.py` exists with `rand_str()` and `utc_now()`
- [ ] Neither test file imports from `pyassorted`
- [ ] `pyassorted` removed from `pyproject.toml` dependencies
- [ ] `poetry lock` succeeds and `pyassorted` is absent from `poetry.lock`
- [ ] `requirements.txt` and `requirements-dev.txt` regenerated
- [ ] All 21 existing tests continue to pass
