# MGT-004: Select.order_by()

## Description

Add `.order_by()` to the `Select` statement builder to support sorting results.
Uses Python's unary minus on `ModelField` to express descending order.

## Expected API

```python
# Ascending (default)
stmt = select(User).order_by(User.name)

# Descending — unary minus on ModelField
stmt = select(User).order_by(-User.created_at)

# Multiple sort keys
stmt = select(User).order_by(User.role, -User.age)
```

## Implementation Notes

- Add `__neg__` to `ModelField` → returns `ModelFieldSort(field, DESCENDING)`
- `Select` stores `_sort: List[ModelFieldSort]`
- `.order_by(*fields)` appends; multiple calls accumulate sort keys
- Passes to pymongo as `find(...).sort([("field", 1), ("field", -1)])`
