# MGT-027: v0.4.0 documentation update

## Description

Update the mkdocs documentation to cover all new v0.4.0 APIs. Add a migration
guide from v0.3.x to v0.4.0 if there are breaking changes.

## Scope

1. **Update `docs/querying.md`** — add sections for:
   - Logical combinators (`or_()`, `and_()`, `not_()`)
   - Null operators (`.is_(None)`, `.is_not(None)`)
   - String operators (`.like()`, `.contains()`, `.startswith()`, `.endswith()`)
   - Range operator (`.between()`)
   - `.distinct()`
   - `ScalarResult` iteration (`for user in result:`)
2. **Update `docs/session.md`** — add sections for:
   - `session.refresh()`
   - `session.merge()`
   - Session state properties (`.dirty`, `.new`, `.deleted`)
3. **Add `docs/indexes.md`** — document `__indexes__` and `create_indexes()`.
4. **Update `docs/index.md`** nav table with new pages/sections.
5. **Add `docs/migration-v0.3-to-v0.4.md`** if any breaking changes exist.

## Acceptance Criteria

- [ ] All new APIs documented with examples
- [ ] `mkdocs build --strict` passes
- [ ] Navigation updated
- [ ] Migration guide added if needed
