# MGT-003: ScalarResult.exists()

## Description

Add `.exists()` to `ScalarResult` that returns `True` if at least one document
matches the statement's filters.

## Expected API

```python
has_admin = session.scalars(select(User).where(User.role == "admin")).exists()
```

## Implementation Notes

- Use `collection.count_documents(filter, limit=1) > 0`
- More efficient than `.count() > 0`

## Acceptance Criteria

- [x] `.exists()` returns `bool`
- [x] Respects `.where()` filters
- [x] Test added to `tests/test_orm.py`
