# MGT-007: Bulk delete via session.execute()

## Description

Introduce a `Delete` statement and a `delete()` factory function so bulk
deletes can be expressed as first-class statements passed to `session.execute()`.

## Expected API

```python
from mongotic import delete

stmt = delete(User).where(User.active == False)
affected = session.execute(stmt)  # returns int (deleted count)
```

## Implementation Notes

- `delete(Model)` → `Delete(orm_model=Model)`
- `Delete` has `.where()` (same as `Select`)
- `session.execute(Delete)` calls `collection.delete_many(filter)`
- Executes immediately (not staged)
- Returns `int` — number of deleted documents
- Distinguish from `session.delete(instance)` which stages a single deletion

## Acceptance Criteria

- [ ] `delete()` importable from `mongotic`
- [ ] `.where()` chainable on `Delete`
- [ ] `session.execute(delete_stmt)` runs `delete_many` and returns count
- [ ] Tests added to `tests/test_orm.py`
